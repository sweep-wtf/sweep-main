The **Sweep Program** is the on-chain Solana smart contract that powers **Sweep**, the wallet cleaning tool.

## Features
- Detect and swap/burn dust tokens
- Track $SWEEP rewards
- Enable AI-powered reinvestments

## Quick Start
```bash
anchor build
anchor deploy
anchor test
```
